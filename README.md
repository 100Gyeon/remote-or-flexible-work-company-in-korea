2. **원격 근무 Rule**
    - 일주일에 3일 회사 출근 권장.
    - 회사와 팀의 동의가 있다면 길게 해외에서 원격근무 가능 (선례: 3달간 남미, 1달간 치앙마이 , 2주간 제주)
